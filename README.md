# StateObject

StoryBakery가 배포하는 Luau 라이브러리입니다.

- 패키지 이름: `storybakery/state_object`
- 엔트리: `init.luau`
- 설치: `pesde add storybakery/state_object`
- 포함 유틸리티:
  - `StateObject.GetCustomStateObjectTypeAnnotation`
  - `StateObject.SaveUtil`

## 테스트

- 의존성 설치: `pesde install`
- 실행: `lune run scripts/run_tests.luau`

필요한 사용 예제와 추가 설명을 채워주세요.
